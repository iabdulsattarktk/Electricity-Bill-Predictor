# Electricity Bill Predictor & Management System ⚡

A C++ console-based application built as a semester project using Object-Oriented Programming concepts.

## OOP Concepts Used

- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- File Handling

## Features

- Add and manage household appliances
- Calculate electricity consumption
- Generate monthly electricity bills (WAPDA tariff slabs)
- Detect high usage appliances
- Provide energy-saving insights
- Support multiple consumers

## How to Run

1. Clone this repository
2. Open terminal in project folder
3. Compile using:

```bash
g++ main.cpp -o ElectricityBillPredictor
```

4. Run:

```bash
./ElectricityBillPredictor
```
